# Shuten.io - Landing Page

This repo contains the code for the landing page for Shuten [https://shuten.io]


## Tech Stack

- TypeScript v5.8
- React v19
- React Router v7
- Vite v6.3
- Tailwind CSS v4.1


## Package Managers

We use [bun](https://bun.sh/) as our package manager.


## Infrastructure

- We deploy to **Vercel**.
